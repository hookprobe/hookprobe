Edge Deployment (Liberty Architecture)

Democratized security - enterprise-grade protection on $75 hardware

  • Hardware: Raspberry Pi 5 ($75), Intel NUC, or any ARM64/x86_64 SBC
  • Minimum: 8GB RAM, 500GB storage
  • Network: 1Gbps Ethernet (2.5Gbps recommended)
  • Cost: $75-$600 hardware investment
  • Security: Hardware fingerprinting (no TPM required!)
  • Benefits: Complete local control, no cloud dependency, GDPR compliant

Liberty Features:

  • Neural resonance authentication via Neuro Protocol
  • Hardware fingerprinting (CPU, MAC, disk, DMI)
  • HTP protocol for NAT/CGNAT traversal
  • Decentralized Security Mesh participant
  • XDP/eBPF DDoS mitigation at kernel level
  • AI-powered threat detection (Qsecbit + Neuro)
  • Works offline - all data stays on-premises

Ideal for MSSPs, validators, and multi-tenant environments
