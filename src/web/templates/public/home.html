Choose Your Deployment

Flexible deployment options for edge devices and cloud infrastructure

Edge Deployment

Perfect for home users, small businesses, and remote sites

  • Hardware: Run on Raspberry Pi, Intel NUC, or any SBC
  • Minimum: 8GB RAM, 500GB storage
  • Network: 1Gbps Ethernet (2.5Gbps recommended)
  • Cost: $150-$600 hardware investment
  • Benefits: Complete local control, no cloud dependency, GDPR compliant

Key Features:

  • XDP/eBPF DDoS mitigation at kernel level
  • AI-powered threat detection (Qsecbit)
  • Automated Kali Linux response
  • 7-POD architecture for complete security
  • Works offline - all data stays on-premises
  • Install Guide
  • Edge Blog Posts

Cloud Deployment

Ideal for MSSPs, enterprises, and multi-tenant environments

  • Hardware: Datacenter servers or cloud VMs
  • Minimum: 32GB RAM, 1TB storage per node
  • Cluster: 3 Frontend + 3+ Backend nodes
  • Capacity: 100-1000+ customer sites per cluster
  • Benefits: Centralized management, multi-tenant isolation, cross-customer threat intelligence

Key Features:

  • Apache Doris multi-tenant backend
  • Centralized web dashboard for all customers
  • 365+ day data retention
  • GPU-accelerated ML training (optional)
  • Cross-customer threat correlation
  • Cloud Guide
  • Cloud Blog Posts

Hybrid Deployment

Best of both worlds: Edge devices for local security + Cloud backend for centralized management

  • Deploy HookProbe edge devices at customer sites
  • Aggregate data to centralized cloud backend
  • Local UI for high-value customers, headless for others
  • Flexible resource allocation based on customer needs
  • Contact for MSSP Setup
